# rock-paper-scissors
A simple and interactive Rock, Paper, Scissors game built with HTML, CSS, and JavaScript. The game allows users to play against the computer, with their scores being tracked and stored using localStorage, ensuring that scores persist across sessions.


## Key Features
- **User vs. Computer Gameplay**: Users can choose between rock, paper, and scissors to play against the computer's random choice.
- **Persistent Score Tracking**: Scores for wins, losses, and ties are saved in localStorage, preserving the game state across browser sessions.
- **Dynamic UI Updates**: The game updates the displayed scores and move results in real-time.
- **Emoji Representation**: Rock, paper, and scissors moves are visually represented using emojis for a fun and engaging experience.
- **Reset Functionality**: Users can reset their scores at any time with the reset button.


## Play the Game:
- Choose your move by clicking the corresponding button.
- View the result and updated scores.
- Use the reset button to clear scores if needed.
